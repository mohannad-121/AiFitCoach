## 🎯 الخطوات السريعة | Quick Steps

### 1. افتح المتصفح وذهب إلى:
```
http://localhost:8080
```

### 2. سجل دخول جديد:
- **البريد الإلكتروني**: أي بريد (مثلاً: `test@example.com`)
- **كلمة المرور**: أي كلمة مرور (6 أحرف على الأقل)
- **الاسم**: أي اسم (خاص بـ Sign Up)
- انقر **دخول** أو **إنشاء حساب**

### 3. استكمل البيانات:
- **الصفحة الرئيسية** ستفتح تلقائياً
- انقر على **Onboarding** أو كمل الإعداد
- أدخل:
  - الوزن والطول (Measurements)
  - **الأمراض المزمنة** ✨ (المضافة حديثاً)
  - **الحساسيات** ✨ (المضافة حديثاً)
  - الأهداف والموقع

### 4. اختبر التنقل:
```
من الشريط العلوي انقر على:
├─ Home (الرئيسية)
├─ Workouts (التمارين)  ← جديد: محمي الآن
├─ AI Coach (المدرب الذكي)
├─ Schedule (الجدول)
└─ Profile (الملف الشخصي)
```

### 5. جرب الدردشة مع AI:
- انقر على **AI Coach**
- ابدأ المحادثة
- سيستلم AI:
  - ✅ اسمك
  - ✅ وزنك وطولك
  - ✅ **أمراضك المزمنة** ✨
  - ✅ **حساسياتك** ✨
  - ✅ أهدافك والموقع

## 🐛 إذا حدثت مشكلة | If Something Goes Wrong

### المشكلة: لا يمكن الذهاب إلى الصفحة الأخرى
```javascript
// افتح F12 وفي Console اكتب:
localStorage.clear()
location.reload()
```

### المشكلة: رسالة تحميل لا تختفي
```javascript
// في Console:
localStorage.getItem('fitcoach_mock_user')
// يجب أن ترى: {"id":"user...", "email":"..."}
```

### المشكلة: الخادم لا يستجيب
```powershell
# تحقق من أن الخوادم تعمل:
netstat -ano | findstr ":8080"
netstat -ano | findstr ":8000"

# إذا لم تراهم، أعد التشغيل:
npm run dev  # في terminal 1
python ai_backend/main.py  # في terminal 2
```

## ❓ أسئلة شائعة | FAQ

**س: هل بيانات المستخدم تبقى بعد الانغلاق؟**
ج: نعم! تُحفظ في localStorage (ستبقى حتى مسح البيانات)

**س: ماذا لو نسيت كلمة المرور؟**
ج: Mock Auth لا يتحقق من الكلمة، أي كلمة تعمل

**س: هل يمكن حذف الحساب؟**
ج: انقر Logout وسجل حساب جديد

**س: كيف أتصل بـ Supabase الفعلي؟**
ج: اطلب في `.env.local`:
```
VITE_SUPABASE_URL=your_url
VITE_SUPABASE_ANON_KEY=your_key
```
